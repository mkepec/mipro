## Student-Suitable Topics for 2nd Year Undergrads (SW Developers)

Based on the analysis, here are suitable project types:

### Recommended Topic Areas (Based on Historical Trends)

1. **IoT/Embedded Projects** (Most common - good for hands-on)
   - Arduino/Raspberry Pi-based systems
   - Sensor data collection and monitoring
   - Smart device implementations

2. **AI/ML Applications** (Growing trend)
   - Using ChatGPT/LLM APIs for applications
   - Simple ML models for classification
   - Image/text processing with existing libraries

3. **Web/Mobile Development** (Strong presence)
   - Web applications with database backend
   - Mobile apps for specific use cases
   - Performance comparisons of frameworks

4. **Performance/Comparative Studies** (Very common)
   - Comparing different technologies/approaches
   - Benchmarking tools/frameworks
   - Evaluation of existing solutions

### Complexity Recommendations

For 2nd year undergrads, target **Medium complexity** papers:
- Full working implementation (not just overview)
- Clear problem statement and solution
- Measurable results (performance, functionality)
- Avoid: Novel algorithms, complex optimizations
- Focus: Practical applications, comparative studies, integrations
